# 📊 Analysis of Reasons for Investment

## 📌 Project Overview

The **Analysis of Reasons for Investment** project is a Power BI dashboard that explores the key factors influencing investment decisions. It presents interactive visualizations and analytical insights to understand investor motivations, preferences, and trends using data-driven techniques.

---

## 🎯 Objectives

* Analyze the primary reasons behind investment decisions.
* Visualize investment trends through interactive dashboards.
* Identify patterns in investor preferences.
* Generate meaningful business insights from financial data.
* Support data-driven decision-making through visual analytics.

---

## 🛠️ Technologies Used

* Microsoft Power BI
* Data Analysis
* Data Visualization
* Business Intelligence

2. Open `Cognifyz Task 5.pbix` in Power BI Desktop.
3. Refresh the data if required.
4. Explore the interactive dashboard.
